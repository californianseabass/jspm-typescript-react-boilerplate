
## How To Use
```bash
git clone https://github.com/californianseabass/jspm-typescript-react-boilerplate.git $my-repo
cd $my_repo
rm -rf .git
npm i
```

### Installation
```bash
npm i
```

### Running
```bash
npm run start
```

### Linting
```bash
npm run lint
```

## App Configuration
Uses dotenv to handle configuration.
